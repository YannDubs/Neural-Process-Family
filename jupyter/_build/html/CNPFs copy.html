Conditional NPFs¶

Graphical model for the Conditional Neural Process Sub-Family.¶

Conditional Neural Process Sub-Family (CNPFs), are NPFs that do not have any latent random variables. They all have the same probabilistic graphical model (graph_model_CNPF), but the computation graph is quite different because of differences in the aggregator \(\mathrm{Agg}\). We will be talking about the following and experimenting with the following:

  • Conditional Neural Process (CNP) [GRM+18]. The aggregator is a simple average.

  • Attentive Neural Process (AttnCNP) [KMS+19] [^AttnCNP]. The aggregator is an attention mechanism.

  • Convolutional Conditional Neural Process [GBF+19]. The aggregator is a set convolution.

The training of CNPFs is straight forward and consists in optimizing a Monte Carlo estimate of {eq:training}.
